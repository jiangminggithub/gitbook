# splitter 侧边栏宽度可调节



`splitter` 插件可以使左侧的`侧边栏目录宽度`可以自定义的调节。

**配置使用方法：**

```json
{
    "plugins": ["splitter"]
}
```

插件 Github 地址：[https://github.com/yoshidax/gitbook-plugin-splitter](https://github.com/yoshidax/gitbook-plugin-splitter)





**效果预览：** 参考本书的左侧边栏，可以调节宽度
