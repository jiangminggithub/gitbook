# copy-code-button 代码复制按钮



为代码块添加一个可以`复制的按钮`。

**配置使用方法：**

```json
{
    "plugins": ["copy-code-button"]
}
```

插件 Github 地址：[https://github.com/WebEngage/gitbook-plugin-copy-code-button](https://github.com/WebEngage/gitbook-plugin-copy-code-button)
