# code 代码添加行号&复制按钮



这个插件可以为代码块添加`行号`和`复制按钮`，单行代码情况无行号。

如果需要`去除代码复制按钮`，可在配置文件进行配置 `copyButtons` 属性为 false。

**配置使用方法：**

```json
{
    "plugins" : [ 
            "code" 
     ],
    "pluginsConfig": {
      "code": {
        "copyButtons": false
      }
    }
}
```

插件 Github 地址：[https://github.com/TGhoul/gitbook-plugin-code](https://github.com/TGhoul/gitbook-plugin-code)



**效果预览：**



```json
{
    1,
    2,
    3,
    ...
}
```
