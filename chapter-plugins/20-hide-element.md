# hide-element 隐藏元素



可以隐藏不想看到的元素，比如导航栏中 `Published by GitBook` 。

**配置使用方法：**

```json
{
    "plugins": [
        "hide-element"
    ],
    "pluginsConfig": {
        "hide-element": {
            "elements": [".gitbook-link"]
        }
    }
}
```

插件 Github 地址：[https://github.com/gonjay/gitbook-plugin-hide-element](https://github.com/gonjay/gitbook-plugin-hide-element)
