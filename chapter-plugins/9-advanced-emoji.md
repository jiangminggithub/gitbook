# advanced-emoji 表情图标



这个插件可以在书籍中使用[表情列表](http://www.emoji-cheat-sheet.com/)中的表情图标。

**配置使用方法：**

```json
{
    "plugins": [
        "advanced-emoji"
    ]
}
```

插件 Github 地址：[https://github.com/codeclou/gitbook-plugin-advanced-emoji](https://github.com/codeclou/gitbook-plugin-advanced-emoji)



**效果预览：**



:smile:

:sunny:

:mouse:

:bell:

:icecream:

:house_with_garden:
